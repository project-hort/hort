Install hort-cli

The Hort command-line client.

Linux / macOS

curl -fsSL https://hort.rs/install-cli.sh | sh

Windows (PowerShell)

irm https://hort.rs/install-cli.ps1 | iex

The installer is fail-closed: it verifies the SHA-256 and the keyless cosign signature of every download before installing — no valid signature, no install. If cosign isn't already on your system the installer bootstraps a version-pinned copy and verifies its checksum first. To verify a download yourself, see the manual verification guide.

Options

Both scripts accept --version vX.Y.Z (pin a release), --dir <path> (install location), and --add-to-path (PowerShell: -AddToPath). Pin a version:

HORT_VERSION=v1.0.0 curl -fsSL https://hort.rs/install-cli.sh | sh

There is intentionally no option to skip verification. Uninstall is a single file: remove hort-cli from your install directory.
